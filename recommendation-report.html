Sub-topic
Three MCP primitives
Tools — model-controlled actions.
Resources — app-controlled read-only data.
Prompts — user-controlled templates.
Study Resource
MCP architecture overview → Course: Intro to MCP →
Hands-on Exercise

Sub-topic
Resource vs Tool
Resources expose read-only data the application decides when to surface. Tools are executable actions the model decides when to invoke.
Study Resource
MCP Resources concept → MCP in Claude Code →
Hands-on Exercise
Given: "fetch a database row" and "display the current file tree" — classify each as Resource or Tool and justify your answer.
Sub-topic
Mid-session server limitation
New MCP servers cannot be hot-loaded into a running session. The session must be restarted for a newly registered server to become available.
Study Resource
Course: MCP Advanced Topics →
Hands-on Exercise
Add a new MCP server to .claude/settings.json while a Claude Code session is open. Confirm the server is unavailable until restart.
