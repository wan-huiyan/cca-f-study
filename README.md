## How to Use

### Taking the Quiz
1. Open `cca-f-practice-quiz.html` in your browser
2. Answer all 30 questions (or filter by domain)
3. Click **"Save Attempt"** — results are saved to browser localStorage
4. Click **"Export Progress JSON"** to download results for Claude to analyse
5. Share the exported JSON with Claude to get updated study recommendations

### Tracking Progress
- **Baseline:** Take the quiz before any studying to establish where you are
- **Weekly:** Retake after each study week to measure improvement
- **Pre-exam:** Final attempt in Week 4 to confirm readiness (target: 80%+)

### Updating the Plan
After each quiz attempt, ask Claude to:
1. Read the exported progress JSON
2. Compare against `progress.json` advantage map
3. Adjust study plan focus based on weak domains
4. Optionally generate new questions for weak areas
